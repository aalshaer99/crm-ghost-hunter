# CRM Ghost-Hunter 
**High-Fidelity Revenue Operations & Data Decay Engine**

###  The Real-World Delta
B2B CRM data is notoriously unreliable. Most organizations suffer from "Data Debt", paying for thousands of "Ghost Leads" who have moved companies or lost budget authority. This tool provides a **Technical Audit** of CRM health, moving beyond simple cleaning into **Predictive Revenue Recovery.**

###  Strategic Features
* **Decay ROI Calculator:** Quantifies the literal "Cloud Tax" (Salesforce/HubSpot storage costs) of maintaining stale data.
* **The Promotion Trigger:** Uses NLP to compare historical titles vs. current API data to identify "Champions" who have gained new budget authority.
* **Engagement Velocity Scoring:** A proprietary logic-gate that separates "Curiosity Clicks" from "Buying Intent."

###  Architectural Proof
This project demonstrates the ability to handle **complex conditional logic**, **date-time mathematics**, and **API-ready JSON structures**—the exact skills needed to manage a $10M+ revenue pipeline.
